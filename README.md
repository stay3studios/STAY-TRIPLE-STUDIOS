# Slot Casino Website Documentation

## Features
- User-friendly interface with vibrant graphics.
- Multiple slot games with varying themes and gameplay mechanics.
- Real-time leaderboard for competitive play.
- In-app rewards and bonuses.

## Installation Instructions
1. Clone the repository: `git clone https://github.com/stay3studios/STAY-TRIPLE-STUDIOS.git`
2. Navigate to the project directory: `cd STAY-TRIPLE-STUDIOS`
3. Install dependencies: `npm install`
4. Start the application: `npm start`

## How to Play
- Select a game from the main menu.
- Set your bet amount.
- Press the spin button to start playing.
- Watch out for winning combinations indicated on the pay lines.
